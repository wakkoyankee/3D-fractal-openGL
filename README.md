# 3D-fractal-openGL

Executer l'executable, puis dans l'invite de commande mettre:
- 1 : pour le projet cpu
- 2 : pour le projet gpu

OU

Commandes : 
python gpu-proj.py
python cpu-proj.py

Parametres changeables (dans le fonction draw de la class colorShader): 

- SCALE
- C_X
- C_Y
- C_Z
- ROT1_X
- ROT1_Y
- ROT1_Z 
- ROT2_X
- ROT2_Y 
- ROT2_Z
